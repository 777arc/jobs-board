# NumFOCUS jobs board

## Installation

Install gatsby:

```shell
npm install -g gatsby
```

Install dependencies:

```shell
npm install
```

## Develop

```shell
gatsby develop
```
The site is now running at `http://localhost:8000`

## Deploy

```shell
gatsby build
```

The site is generated into `./public`.

## Site generator

This site is based on [Gatsby](https://gatsbyjs.com)
